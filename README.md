# Dr.MediaShot - Medical Marketing & Digital Solutions Agency

A high-performance, premium landing page for **Dr.MediaShot** - a leading agency specialized in medical marketing and digital transformations for healthcare professionals. Built with Next.js 14, TypeScript, and Vanilla CSS/Tailwind.

## 🚀 Live Demo
Check out the live version here: [https://khaki-sheep-702294.hostingersite.com/](https://khaki-sheep-702294.hostingersite.com/)

## ✨ Key Features

- ⚡ **Next.js 14 App Router** - Utilizing Server Components for lightning-fast performance.
- 🌍 **Bilingual (AR/EN)** - Full support for Arabic (RTL) and English (LTR) with seamless switching.
- 🎭 **Premium Animations**:
  - Custom slide-in effects for brand identity.
  - Floating and interactive logo animations.
  - Smooth scroll-reveal effects for all sections.
  - Dynamic number counters for agency statistics.
- 📱 **Mobile-First Design** - Fully responsive and optimized for all devices.
- 💼 **Service Showcases** - Interactive displays for medical management systems and websites.
- 🎯 **SEO & Performance** - Optimized meta tags, semantic HTML, and high Core Web Vitals scores.

## 📋 Core Sections

1. **Hero Section** - Premium branding with interactive floating elements.
2. **Announcement Bar** - Dynamic marquee for latest offers and capacity updates.
3. **About Us** - Detailed insight into the agency's mission and team.
4. **Services** - Comprehensive breakdown of medical marketing, branding, and tech solutions.
5. **System & Web Showcase** - Interactive preview of the clinic management systems.
6. **Stats & Trust** - Data-driven section showing agency achievements.
7. **Testimonials** - Real feedback from medical professionals.
8. **Contact & WhatsApp** - Multi-channel communication for lead generation.

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS & Custom CSS Keyframes
- **Icons**: Lucide React
- **Fonts**: Alexandria (Arabic), Playfair Display (English)
- **Deployment**: Optimized for Vercel/Hostinger

## 📦 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```
   Access at [http://localhost:3000](http://localhost:3000)

## 🎨 Design System

The project uses a sophisticated **Purple & Gold** palette:
- **Primary**: Deep Purple (`#0A0118`)
- **Accent**: Golden Amber (`#FFB800`)
- **Glassmorphism**: Backdrop blur effects with thin borders for a premium feel.
